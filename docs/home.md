### .doFetchHomeBanner([obj])

Retrieve Home banner info

**Dispatch**: `FETCH_HOME_BANNER`

**Returns**: {Promise}

**Category**: actions  

| Param | Type | Description |
| --- | --- | --- |
| [obj] | {Object} | Accept the following: |
| [obj.bannerId] | {String} | Id of the banner to retrieve the info from |

**Example**  

```js

```
